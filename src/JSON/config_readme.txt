prefix - стандартный префикс бота
token - токен бота
owners - список айди владельцев
embColor - стандартный цвет эмбедов
antibot - будет ли бот реагировать на других ботов
support - айди канала для команды support
logger, errlogger - логгеры для использованных команд и ошибок (вебхуки)
sdc, boticord - мониторинги

после настройки переименуйте config_example.json в config.json
